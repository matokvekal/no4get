# Component Numbers Reference

Numbers appear as red badges on hover, or always-on with the **# OFF / # ON** toggle button (bottom-right corner).  
Keyboard shortcut: **Ctrl+Shift+D**

To delete this system later: remove `DevLabel` wrapper from each file listed, delete `src/components/DevOverlay/`, and revert `App.tsx`.
